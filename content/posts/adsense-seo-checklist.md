## 1. 콘텐츠 품질

애드센스는 **독창적이고 유용한 콘텐츠** 를 가장 중요하게 봅니다.

- 복사·붙여넣기 글 지양
- 한 주제를 깊이 있게 다루기
- 최소 800자 이상의 정보성 글 권장

## 2. 기술적 SEO

| 항목 | 상태 |
| --- | --- |
| 고유 title / description | Metadata API 적용 |
| sitemap.xml | 빌드 시 자동 생성 |
| robots.txt | 빌드 시 자동 생성 |
| 모바일 반응형 | Tailwind 반응형 레이아웃 |

## 3. 사용자 경험

- 읽기 편한 폰트와 충분한 여백
- 다크 모드 지원
- 빠른 페이지 로드 (SSG)

<!-- ad-break -->

## 4. 필수 페이지

애드센스 신청 전 아래 페이지를 준비하세요.

- **개인정보처리방침**
- **이용약관** (선택)
- **문의하기** 또는 About 페이지

## 5. 서치 콘솔 연동

1. [Google Search Console](https://search.google.com/search-console) 에 도메인 등록
2. `/sitemap.xml` 제출
3. 색인 생성 상태 모니터링

꾸준히 양질의 글을 발행하면 승인 가능성이 높아집니다.
