## Editorial Overview

> As an independent tech review publication, we match air conditioners to **room use patterns** rather than BTU labels alone. For summer {{today}}, ceiling height, sun exposure, and whether you sleep or work in the space change the right install type more than brand marketing.


## Introduction: one size does not fit every room

Air conditioner shopping spikes when heat advisories hit — but a unit that works in a shaded bedroom may fail in a west-facing studio with a kitchen nook. This scenario guide maps **three common apartment layouts** to realistic picks using public specs and listed prices as of {{today}}.

## Scenario: Shaded bedroom (12–15 m²)

**What matters**

- Night noise below ~53 dB for light sleepers
- Stable temperature without wide swings
- Minimal light leak from window install

**Recommended pick:** **8,000 BTU window unit** (e.g., Frigidaire FHWW083WBE class) — ~$249 / ~{{krw:249}}

**Runner-up:** 10,000 BTU single-hose portable if window install is banned

**Why:** Bedrooms need efficiency over 8-hour runs; window units avoid single-hose warm-air draw from the corridor.

| Factor | Target |
| --- | --- |
| BTU | 7,000–9,000 |
| Install | Window preferred |
| Noise | ≤53 dB |

---

## Scenario: West-facing studio (20–25 m²)

**What matters**

- Higher BTU headroom for afternoon sun load
- Dehumidify during humid weeks
- Ability to cool combined sleep + desk zone

**Recommended pick:** **14,000 BTU dual-hose portable** (LG LP1419IVSM class) — ~$549 / ~{{krw:549}}

**Runner-up:** 12,000 BTU window unit if a large sash window is available

**Why:** Open studios punish under-sized portables; dual-hose keeps pressure balanced when the kitchen and desk share air.

| Factor | Target |
| --- | --- |
| BTU | 12,000–14,000 |
| Install | Dual-hose or large window |
| Extra | Dehumidify mode |

---

## Scenario: Home office corner (10–12 m² within larger unit)

**What matters**

- Daytime noise tolerance higher than bedroom
- Quick cool-down before video calls
- Option to move unit if lease ends

**Recommended pick:** **10,000 BTU single-hose portable** (Midea MAP10S1TBL class) — ~$379 / ~{{krw:379}}

**Runner-up:** 8,000 BTU window on office window only

**Why:** WFH users often cool **one zone** for 6–8 hours, not the whole apartment — portables win on flexibility if the main bedroom already has AC.

| Factor | Target |
| --- | --- |
| BTU | 9,000–10,000 |
| Install | Portable OK |
| Priority | Mobility |

---

## Scenario: Rental with strict window rules

**What matters**

- No permanent brackets; balcony-door exhaust kits
- Landlord-friendly install photos
- Accept higher electricity vs central building AC

**Recommended pick:** **Dual-hose portable with door seal kit** — prioritize 12,000+ BTU

**Runner-up:** Evaporative cooler only in dry climates (not a full AC replacement)

**Why:** When policy blocks window hardware, **hose sealing quality** matters more than brand; dual-hose reduces fighting the building’s warm hallway air.

| Factor | Target |
| --- | --- |
| BTU | 12,000+ |
| Install | Door/balcony exhaust |
| Avoid | Undersized single-hose for whole unit |

## Final Verdict

Match **install permission first**, then **room use**, then BTU label. Bedrooms favor quiet window units; sun-heavy studios need capacity and dual-hose physics; WFH corners can save money with a smaller portable aimed at one zone.

### Who should buy?

| Situation | Buy |
| --- | --- |
| Sleep-quality priority | 8k window for bedroom |
| One-room living | 14k dual-hose |
| Partial-day desk cooling | 10k single-hose |

### Who should skip?

| Situation | Skip |
| --- | --- |
| Whole studio on 8k window | Under-powered |
| Single-hose for 25 m² all day | Overworked unit |
| Window AC where lease forbids it | Compliance risk |
